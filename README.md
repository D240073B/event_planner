# Event Planner AI

Event Planner AI is a Flutter application that helps users plan events using AI. It generates event posters using Hugging Face Stable Diffusion and provides detailed planning suggestions using Firebase AI Logic (Gemini).

## 🚀 Getting Started

To get the app up and running, please refer to our detailed **[User Guide](user_guide.md)**.

### Features
- **AI Poster Generation**: Create stunning event posters based on your theme.
- **Smart Suggestions**: Get detailed planning advice, including agendas and food recommendations.
- **Easy Input**: Simple form to capture all your event details.
